# 📖🌱일기로운 슬기생활🌱📖

<br/>



-   일기로운 슬기생활은 감정 기반 AI 일기 컨텐츠화 서비스입니다.
-   쉽게 일기를 작성하고 그 일기에 대한 여러 컨텐츠들을 생성하여 제공하는 서비스입니다.

<br/>
